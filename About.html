About the project

This site was created as a university project for a course titled "Internet Jezici i alati 1", where the goal was to build a functional website using HTML and CSS.

Design

The overall design and some of the illustrations on the site were created using Figma, a popular design tool.

I have also sourced additional illustrations from undraw.co, a website that offers a wide range of beautiful, customizable illustrations.

The photos used on the site are all royalty-free and were sourced from pexels.com.

Code

Since the goal of the project was to use only HTML and CSS, some functionality on the site, such as search, login, and registration, is mimicked using these languages.

Code editor used is Visual Studio Code.

The website is fully responsive and is created with mobile first principles in mind.

Additional Notes

Please note that all of the listed apartments and their descriptions on this site are entirely fictional and any resemblance to real listings is purely coincidental. I created these listings as part of the project and they are not based on any real properties.

I hope you enjoy browsing the site and appreciate any feedback you may have. Thank you for visiting and have a great day!
